How to Use It


1. Getting a Random Fact

Send a GET request to /randomfact endpoint


Code Example (Python)

import requests

headers = {
"Content-type": "application/json"
}

response = requests.get(
'http://127.0.0.1:5000/randomfact'
)

# You can now print your response
print(f'response: {response}')

2. Getting All Facts

Send a GET request to /factbase endpoint

import requests

headers = {
"Content-type": "application/json"
}

response = requests.get(
'http://127.0.0.1:5000/factbase'
)

# You can now print your response
print(f'response: {response.json()}')
